This app is used to determine if two user uploaded images are identical. Please note that the images must have the same
dimensions for them to be considered identical. 

Instructions:
> Run the code
> Navigate to File -> Upload Image to upload your first image
> Select desired image and choose if it will be image 1 or image 2
> Repeat the process for the other image
> Click on "Test" button to determine if the two images are the same
> If you want to choose a new image, repeat the steps
> If you want to clear both images, navigate to File -> Reset

Instructions available via Help -> Instructions


This is the successor to hotDogTest (found at https://github.com/reedfodge/hotdogtest)
